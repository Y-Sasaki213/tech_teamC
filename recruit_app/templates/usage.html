候補者一覧に戻る

このアプリについて

この新卒採用管理ツールでは、候補者の登録、一覧表示、詳細確認、編集、削除ができます。 また、面接日や最終更新日から7日以上経過している場合にアラートを表示します。

1. 候補者を登録する

  1. 上部メニューの「新規登録」をクリックします。
  2. 採用者氏名、担当者、連絡ステータスを入力します。
  3. 必要に応じて、一次面接日、二次面接日、最終面接日などを入力します。
  4. 「登録する」ボタンを押すと、候補者情報が保存されます。

2. 候補者を検索する

  1. 候補者一覧画面を開きます。
  2. 採用者氏名または担当者名を入力します。
  3. 検索ボタンを押すと、条件に一致する候補者だけ表示されます。
  4. 入力欄を空にすると、全件表示できます。

3. アラートを確認する

一覧画面で赤く表示されている候補者は、対応が必要な可能性があります。

  • 一次面接から7日以上経過して、二次面接日が未入力の場合
  • 二次面接から7日以上経過して、最終面接日が未入力の場合
  • 日付項目から7日以上経過している場合
  • 最終更新から7日以上経過している場合

4. 候補者情報を編集する

  1. 一覧画面または詳細画面から「編集」をクリックします。
  2. 変更したい項目を修正します。
  3. 「更新する」ボタンを押すと、情報が更新されます。

5. 候補者を削除する

一覧画面の「削除」ボタンを押すと、候補者情報を削除できます。 削除したデータは元に戻せないため、注意してください。
